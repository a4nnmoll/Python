# 🐍 Python Lab Experiments (BCA)

## 📌 Project Description
This repository contains all Python lab experiments performed as part of the BCA curriculum.  
The aim of this project is to understand basic to intermediate Python concepts through practical programs.

---

## 📚 Experiments Included

### ✅ Experiment 1: Python Installation and Basics
- Python installation
- Print statements
- Variables and data types

### ✅ Experiment 2: Input Statements and Operators
- Taking input using input()
- Type conversion
- Arithmetic, comparison, and logical operators

### ✅ Experiment 3: Conditional Statements
- if, if-else, if-elif-else
- Even/odd, grading system, largest number

### ✅ Experiment 4: Loops
- for loop, while loop
- Sum of numbers
- Multiplication table
- Patterns
- break and continue

### ✅ Experiment 5: Strings and Sets
- String operations and methods
- Vowel counting
- Set creation and operations

### ✅ Experiment 6: Lists, Tuples, Dictionary
- List operations
- Tuple properties
- Dictionary operations
- Traversing collections

### ✅ Experiment 7: Functions
- Function definition and calling
- Parameters and return values
- Factorial and prime number programs
- Local and global variables

---

## ⚙️ Requirements
- Python 3.x installed
- Jupyter Notebook / VS Code / any Python IDE

---

## ▶️ How to Run
1. Install Python from https://www.python.org  
2. Open project in VS Code / Jupyter Notebook  
3. Run the `.ipynb` or `.py` files  
4. Check output in terminal/notebook  

---

## 🎯 Learning Outcomes
- Basic Python syntax
- Problem-solving using Python
- Understanding of data types and control structures
- Writing reusable code using functions
